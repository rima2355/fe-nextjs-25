Metode DOM yang Digunakan

document.createElement()

Digunakan untuk membuat elemen HTML baru secara dinamis.

appendChild()

Menambahkan elemen baru sebagai anak dari elemen parent.

remove()

Menghapus elemen dari DOM.

style.property

Mengubah properti style elemen (seperti backgroundColor).

textContent

Mengubah atau membaca konten teks dari elemen.

classList.toggle()

Menambah atau menghapus kelas CSS dari elemen (toggle).
